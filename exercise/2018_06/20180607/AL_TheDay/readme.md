**It is used to get how many days in the year by dates**  
通过日期YYYY/MM/DD的格式来判断  
这一天属于这一年的第几天  

Problem Description

给定一个日期，输出这个日期是该年的第几天。

 

Input

输入数据有多组，每组占一行，数据格式为YYYY/MM/DD组成，具体参见sample input ,另外，可以向你确保所有的输入数据是合法的。

 

Output

对于每组输入数据，输出一行，表示该日期是该年的第几天。

 

Sample Input

```
1985/1/20
2006/3/12
```

 

Sample Output

```
20
71
```
